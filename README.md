# collaboreats

## Features
 - Create room
    - Title
    - Location
    - Date/time (?)
    - Expiry (1 hour, 2 hours, 12 hours, 24 hours, 48 hours)
 - Join room
    - Via link
 - Sessions for users
 - Nicknames for users
 - Search for a restaurant
 - Add restaurant to room
 - Vote on restaurant
 - Information about restaurant (Yelp link, Google Maps link, rating, etc.)

## Technology used
 - Node.js
    - Express.js
    - Socket.IO
 - MongoDB
 - Vue.js

## Routes
 - Homepage (`/`)
    - Create room
 - Room (any string)
    - Set username
    - Search for restaurant
    - Add restaurant
    - Vote restaurant
